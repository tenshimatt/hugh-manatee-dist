# ADR: Excel Auto-Population for Budget Estimates

**Status:** proposed

## Context
The architectural team currently uses a large Excel file for budget estimation and bill of materials. There is an opportunity to integrate the AES (Architectural Estimation System) with drag-and-drop functionality to automatically pre-populate budget estimates. This would streamline the estimation process and reduce manual data entry errors. The current workflow involves manual population of estimates, which is time-consuming and prone to errors. The team wants to explore automation capabilities to improve efficiency in the architectural estimation phase.

## Decision
Implement Excel auto-population functionality for budget estimates by integrating AES with drag-and-drop capabilities. The system will allow users to drag AES components directly into the Excel estimator, which will automatically populate relevant budget fields and bill of materials. This integration will be developed as part of the broader workflow automation initiative, focusing on the architectural side before processing phases.
